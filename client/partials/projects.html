Ping Pong

     A ranking application for Coding Dojo students that play ping pong more than they code. Students have a global rank and their current cohort rank. This application has a special feature where students can challenge any students via text as long as they have a current valid phone number in the system. Future updates will include a chat room and tournament games. Download this application prior to attending Coding Dojo ad climb the ranks before it's too late!

Doctor Appointment

     Feeling ill? This application will set appoinment to avaible doctors in the area. These doctors will only see four patients a day to insure a good relationship. Doctors on this application are only to treat and look out for your health. Little problems to severe illness, download this app to get the best quality of care.

Stock Screener

     Get rich quick? There is only one way, invest in stocks! This application allows users to follow stocks on the S&P500. Users can see their stocks profit margins for the day and see previous history on a dynamic illustrated graph! Get this app, learn the trends, and get rich quick!

Ninja Gold
